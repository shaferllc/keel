# keel-mcp

Published entry for Keel's [MCP](https://modelcontextprotocol.io) server.

```bash
# One-shot installer (writes .mcp.json)
curl -fsSL https://keeljs.com/install.sh | bash

# Or via npm
npx -y keel-mcp@latest init
```

This package depends on [`@shaferllc/keel`](https://www.npmjs.com/package/@shaferllc/keel) and forwards to its `keel-mcp` bin.
